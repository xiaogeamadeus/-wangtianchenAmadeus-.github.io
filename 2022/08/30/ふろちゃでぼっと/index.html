ふろちゃでぼっと / Flowcha-de-bot

コードが書けなくても、フローチャートを組み立てる要領で簡単にLINE Botが作れる、小学生向けのビジュアルプログラミングWebアプリです。

A visual programming web application for elementary school students that makes it easy to create a LINE Bot by following a flowchart, even if you don’t know how to write code.
