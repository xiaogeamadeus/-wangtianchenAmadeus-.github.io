The Way to Input/Output File by Java

注：本文仅为个人学习笔记，无任何版权。

打开并读取文件对于大多数编程语言来说是非常实用的。Java8与java7新增的java.nio.file包以及streams与文件结合使得文件操作编程变得十分简单。

对于文件操作来说，最基本的两个组件是

  1. 文件或者目录的路径；
  2. 文件本身。

文件和目录路径

一个Path对象表示一个文件或者目录的路径，可以跨操作系统和文件系统。
